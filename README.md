# Google Ads Campaign Project

## Overview
This project demonstrates planning, creating, and analyzing a Google Search Ads campaign.

## Structure
- **plan/** → Keyword research and campaign planning
- **create/** → Draft ad copy and creatives
- **analysis/** → Performance tracking and reports

## How to Use
1. Review `plan/keywords.md` for keyword strategy.
2. Check `create/adcopy.md` for ad copy drafts.
3. Analyze `analysis/report.md` for campaign performance.
